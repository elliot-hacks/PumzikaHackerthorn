📋 Overview

Pumzika Hackerthorn is an AI-powered review analytics platform designed specifically for the East African hospitality market. It automatically analyzes guest reviews to extract sentiment, topics, and actionable insights using a sophisticated hybrid approach.

🌍 Multilingual Support

Full sentiment analysis for Swahili and English reviews with code-switching detection

🤖 AI-Powered

AfriSenti transformer model + LLM integration for accurate sentiment analysis

📊 Real-Time Analytics

Live dashboards with Chart.js visualizations and instant query responses

💡 Actionable Insights

AI-generated narratives with strengths, weaknesses, and recommendations

✨ Key Features

Sentiment Analysis Topic Extraction Aspect Scoring Swahili NLP Command Palette AI Insights Batch Processing Celery Tasks

Core Capabilities

  • Sentiment Classification: Positive, Negative, Neutral with confidence scores (0.0-1.0)
  • Language Detection: Automatic detection of English, Swahili, or mixed language reviews
  • Topic Extraction: 13 canonical hospitality topics (Cleanliness, Staff, Location, etc.)
  • Aspect Scoring: 8 hospitality aspects scored from 0% to 100%
  • Key Phrase Extraction: Important phrases and quotes from reviews
  • AI Narratives: Generated summaries with strengths, weaknesses, and actionable advice

🗃️ Data Models

Model Description Key Fields
Review Individual guest reviews with NLP analysis results sentiment, sentiment_score, topic_labels, aspect_scores, language
TopicCluster Discovered topic clusters with statistics label, review_count, avg_sentiment_score, keywords
SentimentSnapshot Daily aggregate sentiment per property property_id, snapshot_date, positive_count, avg_sentiment
PropertyInsight AI-generated narrative insights for properties strength_summary, weakness_summary, actionable_advice, overall_narrative

⚙️ NLP Pipeline

The platform uses a sophisticated multi-stage pipeline:

1. Language Detection

Uses Swahili marker words and langdetect to identify review language (EN/SW/Other)

2. Sentiment Scoring

  • Swahili: AfriSenti transformer model with lexicon fallback
  • English: Heuristic keyword matching (batch) or LLM (on-demand)
  • Fallback: Reviewer score (1-10) as strong signal

3. Topic Extraction

Maps review content to 13 canonical hospitality topics using keyword matching and LLM

4. Aspect Scoring

Scores 8 aspects (cleanliness, staff, location, value, amenities, wifi, food, noise) from 0-100%

🛠️ Management Commands

Data Ingestion

  • python manage.py ingest_kaggle --csv-path /path/to/data.csv - Import Kaggle reviews
  • python manage.py ingest_afrisenti --file-path /path/to/data.tsv - Import AfriSenti data

NLP Processing

  • python manage.py analyze_sentiment --batch-size 200 - Process unprocessed reviews
  • python manage.py extract_topics --update-clusters - Extract topics from reviews
  • python manage.py generate_insights --all - Generate AI insights for properties
  • python manage.py build_snapshots --all - Build daily sentiment snapshots

Testing & Status

  • python manage.py test_afrisenti --text "Hoteli hii ni nzuri" --language sw
  • python manage.py nlp_status - Show system status and statistics
